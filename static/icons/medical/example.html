Ejemplos de Uso

1. Como imagen HTML

<img src="/static/icons/medical/stethoscope.svg" alt="Médico" width="24" height="24">

2. Como SVG inline (mejor para personalización)

<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" style="width: 24px; height: 24px; color: #667eea;"> <!-- contenido del SVG --> </svg>

3. En CSS como background

.medical-icon { background-image: url('/static/icons/medical/stethoscope.svg'); background-size: contain; width: 24px; height: 24px; }

4. Cambiar color con CSS

.medical-icon { filter: brightness(0) saturate(100%) invert(49%) sepia(84%) saturate(2417%) hue-rotate(217deg) brightness(94%) contrast(94%); /* Esto cambia el color a #667eea */ }
